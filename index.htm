WING-MAN

Wing-Man is a ease of use Valorant companion Discord-Bot! It is intended to be usable without any user configuration, as to allow for simple quick queries in to riots data-base.

You can count on Wing-Man to always pull the data from Riot in real time, with a ever updating first party data-base.

RIOT-API

Wing-Man fetches requests to the offical Riot-API to gather its data and information on players. This allows for the most up-to date data on players.

SERVER PERMISSIONS

Wing-Man needs very minimal permissions, only requesting the ability to create slash commands.
